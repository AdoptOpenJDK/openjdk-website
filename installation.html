Installation

Linux and macOS installation

  1. Download the latest tar.gz from the Latest release page to a directory that will not move or be deleted.
  2. Using Terminal, navigate to the location of this tar.gz: 
    cd <path/to/your/directory>
  3. Extract the tar.gz. You can use the following command: 
    tar -xf <filename>.tar.gz
  4. Add this version of Java™ to your PATH: 
    export PATH=$PWD/j2sdk-image/bin:$PATH
  5. Check that it has installed correctly: 
    java -version

Windows installation

  1. Download the latest .zip from the Latest release page to a directory that will not move or be deleted.
  2. Using the command prompt, navigate to the location of this .zip: 
    cd <path\to\your\directory>
  3. Extract the .zip. You can use the following command: 
    unzip <filename>.zip
  4. Add this version of Java™ to your PATH: 
    set PATH=%cd%\j2sdk-image\bin;%PATH%
  5. Check that it has installed correctly: 
    java -version
